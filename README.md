# Reveal on Vue.js Template

This is a simple application that allows you to create presentations in Vue.js using Reveal.js.

- Vue.js web app
- Reveal.js for presentations
- Markdown for slides
- Use components for slides

## Getting Started

``` bash
# install dependencies
npm install

# serve
npm run dev
```

## Usage

To create your own presentation, you can use the documentation of [Reveal.js](https://revealjs.com/#/).

The big difference is that you can use Vue.js components as your slides to make your code a lot cleaner.

Feel free to customize this template according to your project's specific details and requirements.
